# Get the library

Usage via [unpkg.com](https://unpkg.com/):
```html
<!-- As UMD package -->
<script src="https://unpkg.com/@chialab/dna" type="text/javascript"></script>

<!-- As module -->
<script type="module">
    import('https://unpkg.com/@chialab/dna?module')
        .then((DNA) => {
            //
        });
</script>
```

Install via NPM:
```sh
# NPM
$ npm install @chialab/dna
# Yarn
$ yarn add @chialab/dna
```
